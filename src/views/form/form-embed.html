Embed URL: {{ projectUrl }}/{{ form.path }} {Preview JSON}

This is the url that can be used to embed this form into your application. Your form exists as a JSON object that defines how to render the form.

Prerequisites

If you are using one of our starter kits, all the prerequisits are already set up.

Include the ngFormio module

Be sure you have added the ngFormio library to your application and have included it in the modules list in the app declaration.


            angular.module('yourApp', [
              'formio'
            ]);

Embedding the formio directive

Embed within a template

To embed this form, simply place this directive anywhere in a template. Please note that there are single quotes inside the double quotes. This is since the src of the form is interpolated by angular. You can use a variable as well.


            <formio src="'{{ projectUrl }}/{{ form.path }}'" />
Additional options

Vue.JS

Creating a Vue component

Be sure you have added the formiojs library to your application and imported it in the app declaration.

Embed within a template

To embed a form within your Vue application, lets create a component call FormVue, first you have to import FormioForm from the formiojs library.


            // FormVue.vue
            <template>
                <div id="formio"> </div>
            </template>
            <script>
              import FormioForm from 'formiojs/form';
                export default {
                  name: 'formio-vue',
                  props: ['src'],
           mounted () {
                  let self = this;
               let form = new FormioForm(document.getElementById('formio'));
              form.src = self.src;
              form.on('submit', (submission) => {
                 console.log('The form was just submitted!!!');
            });
             form.on('error', (errors) => {
                  console.log('We have errors!');
            })
            }
        }
    </script>

            ---
            // Then we can use our component in our App.vue.
                  <template>
               <div id="app">
               <formio-vue src="'{{ projectUrl }}/{{ form.path }}'"></formio-vue>
             </div>
            </template>
            <script>
        import FormioVue from './components/Formio.vue'
        export default {
            name: 'app',
            components: {
                FormioVue
          }
        }
     </script>

React

Include the react-formio library

Be sure you have added the react-formio libary to your application.

Render the directive

To embed a form within your react application, import the Formio library and add it to the render tree.


            import React from 'react';
            import ReactDOM from 'react-dom';
            import { Formio } from 'react-formio';

            ReactDOM.render(
              <Formio src="{{ projectUrl }}/{{ form.path }}" />, document.getElementById('example')
            );

Angular 2.x

Include the ng2-formio module

Be sure you have added the ng2-formio library to your application and have included it in the modules list in the app declaration.

Embed within a template

To embed a form within your angular 2.x application, add the formio directive within a template.


            import { FormioModule } from 'ng2-formio';
            @NgModule({
              imports: [ BrowserModule, ReactiveFormsModule, FormioModule ],
              declarations: [ AppComponent ],
              bootstrap: [ AppComponent ]
            })
            export class AppModule { }

            ---

            <formio src='{{ projectUrl }}/{{ form.path }}' />

Quick Embed - iFrame

The iframe embed allows a self contained embed of this form into your application. Use this if you are embedding this within an existing CMS such as Wordpress or Drupal.

With this method, you will not have CSS access to the rendered form.
